<h1 align="center"><b>BIPOC RESOURCES</b></h1>
This program is tailored to the BIPOC community, not only in assisting them with their
mental health but also providing educational information regarding their history.<br>
<p align="center"><b>FEATURES:</b></p>  

* Mental Health Resources  
* Simulated Support Groups
* Diversity Index Calculation
* Event Calendar
* Historical Figure Lookup

<h3 align="center">Mental Health Resources</h3>
<hr>
<p> Includes a diverse range of mental health resources tailored for the BIPOC community. This includes safe space forums, articles/information and assistance with finding a provider.</p>                  
<h3 align="center">Simulated Support Groups</h3>
<hr>
<p>Fully simulated BIPOC support groups that the user may join. Included are names, member counts, and unique descriptions.</p>
<h3 align="center">Diversity Index Calculation</h3>
<hr>
<p>Using the Simpson Diversity Index formula, this program calculates the percentage chance that 2 people will be different races if selected at random.</p>
<h3 align="center">Event Calendar</h3>
<hr>
<p>The user may input a month as an integer and the program will output a BIPOC related event that happened in that month, sometime in history.</p>
<h3 align="center">Historical Figure Lookup</h3>
<hr>
<p>Provides the user with a list of BIPOC historical figures, in which the user may choose which one they would like to learn more about.</p>
